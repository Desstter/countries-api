# Countries API

Explorador de países que consume la [REST Countries API](https://restcountries.com/). Busca cualquier país del mundo y consulta su información detallada: capital, población, región, idiomas y moneda.

## Stack

JavaScript · HTML · CSS

## Features

- Búsqueda de países por nombre
- Filtro por región (África, América, Asia, Europa, Oceanía)
- Vista detallada: bandera, capital, población, idiomas, monedas y países fronterizos
- Modo claro/oscuro
- Diseño responsivo

## Setup

Abre `index.html` directamente en el navegador o sirve con un servidor estático:

```bash
npx serve .
```
